📖 Overview

Community Medicine Toolkit is a standalone web application designed to perform essential epidemiological and clinical calculations commonly used in public health, epidemiology, and community medicine training.

The application consolidates several key population health indicators and clinical measures into a single interactive calculator interface.

It is particularly useful for:

Medical students
Public health professionals
Epidemiology researchers
Community medicine practitioners
Healthcare educators

The application runs entirely inside a single HTML file, requiring no installation or external dependencies.

🔬 Epidemiological Formulas
Body Mass Index (BMI)

BMI = Weight (kg) / Height (m²)

Attack Rate

Attack Rate = (New Cases / Population at Risk) × 100

Crude Mortality Rate

Crude Mortality Rate = (Deaths / Population) × 1000

Case Fatality Rate (CFR)

CFR = (Deaths from Disease / Confirmed Cases) × 100

Sensitivity and Specificity

Sensitivity = TP / (TP + FN)

Specificity = TN / (TN + FP)

Number Needed to Treat (NNT)

ARR = CER − EER

NNT = 1 / ARR
